# FAQ

### 英語を習得するにはどれくらい時間がかかりますか？

非常に長い時間がかかります。

順調に学習を進めた場合、4 年ほどで流暢さに到達できるという意見もあります。「4 年あれば流暢になれる」と考えることは、初心者によくある失敗――短期的な発想によって無理をしすぎて挫折すること――を防ぐ助けになります。
例えば、「単語学習が進むのが遅い」と感じて Anki の新規カード枚数を 1 日 100 枚に設定し、1 週間後に復習量に圧倒されてやめてしまう、といったケースです。

### この表現の意味がわかりません

調べたい語句を X とすると、
「X 意味」
「X とは」
「X 使い方」
などで検索してください。

### 読書素材はどうやって探せばよいですか？

以下のような方法があります。

- 英語タイトルやジャンル名で検索する
- 「free ebook」「public domain」「English graded reader」などのキーワードを併用する
- Project Gutenberg などの無料電子書籍サイトを利用する
- 学習者向けリーダーシリーズを探す

また、学習リソースをまとめたページも参考になります。

### x と y の違いは何ですか？

Google で
「x と y の違い」
「"x" "y" difference」
などと検索すると、多くの場合答えが見つかります。

見つけた説明が理解できない場合、現時点ではその違いを気にする段階ではない可能性があります。その場合は、語彙よりも文法力の向上を優先してください。

### いつから読書を始めるべきですか？

文法ガイドを一通り読み終えたら、すぐに読み始めて構いません。
語彙が 1000 ～ 2000 語程度あると多少楽になりますが、それ以上待つ必要はありません。どのみち最初の読書は難しく感じるものです。

### ネイティブスピーカーはどれくらいの語彙数を知っていますか？

- 小学生レベル：5,000 ～ 20,000 語
- 中学生レベル：20,000 ～ 40,000 語
- 高校生レベル：40,000 ～ 45,000 語
- 大学生レベル：45,000 ～ 50,000 語

※ここでの「語彙数」は辞書に載る基本形のみを数えたものです。固有名詞や複合語は含まれません。

なお、「十分な理解のためにどれくらいの語彙数が必要か」という別の観点の研究もあります。ただし、Anki で学習する必要がある語数と完全に一致するわけではありません。明らかな意味の単語や推測できる単語も多いためです。

### 同じ意味・同じ使い方に見える単語が複数あります。どう区別すればよいですか？

英語の語義が同じに見える場合でも、英英辞典で調べると使用場面やニュアンスの違いが説明されています。また、Google で用例検索するのも有効です。

説明が理解できない場合は、語彙力よりも文法力や読解力を強化してください。
英和辞典だけを見て「同じ意味だから区別できない」と感じる場合、まず辞書の情報が十分でない可能性を疑いましょう。

### Duolingo について

Duolingo の英語コースは、学習量・内容ともに限定的であり、本ガイドで紹介している学習法の代替にはなりません。
文法練習が好きであれば補助的に使っても構いませんが、あくまで副教材として考えてください。
